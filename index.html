WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Langauage. HTML is the content of a web page. Examples of HTML include headings, paragraphs, images, links, list, etc.

Check out the HTML Tutorial on w3schools.com

Learn CSS

CSS stands for Cascading Style Sheets. CSS is the style of a web page. Examples of css include font style and color, background color, borders, dimensions, layout, etc

Check out the CSS Tutorial on w3schools.com.

Learn JavaScript

JavaScript is the progarmming language of the web. JavaScript helps to make web page interactive and dynamic. some common JavaScript task are:

  • add new HTML to the page,
  • change the existing content / style,
  • react to user events (mouse clicks, key presses, etc.)

Check out the JavaScript Tutorial on JavaScript.info.
